# 📊 Post Explorer Dashboard

A responsive React dashboard application that demonstrates UI design, state management, and API integration.

---

## 🚀 Features

- 🔐 Mock Authentication (Login system)
- 📡 API Integration (Fetch posts)
- 🔍 Search functionality
- 🧩 Filter by user
- 📊 Sort by title and ID
- 📄 Pagination
- 📱 Fully responsive design
- ⏳ Loading & error handling
- ♻️ Reusable components

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS (Flexbox & Grid)

---

## 🌐 API Used

https://jsonplaceholder.typicode.com/posts
